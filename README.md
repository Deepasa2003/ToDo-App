# To-Do List App (React)

This is a simple **To-Do List application** built using **React (Vite)** as part of **Module 3 React Assignment**.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Edit existing tasks
- Component-based structure
- State and props management using React hooks

## Technologies Used
- React
- Vite
- JavaScript (ES6)
- CSS
